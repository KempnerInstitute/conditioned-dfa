# Reproducing the results

Every numerical claim in the paper is regenerated by a script in `experiments/`
(or a `slurm/*.sbatch` array), aggregated by a script in `analysis/`, and turned
into a figure by `drafts/Info-DFA/scripts/make_iclr_figures.py`. Aggregate CSVs
land under `results/` (gitignored; sizes are large). This file lists the exact
command, grid, and aggregator for each result.

## Environment
- Python via the `vdc_paper_v2` conda env (torch 2.9, torchvision 0.24, timm 1.0,
  numpy, pandas, scipy). Activate before any run:
  `conda activate /n/holylabs/LABS/kempner_dev/Users/hsafaai/conda_envs/vdc_paper_v2`
- SLURM scripts set `REPO_ROOT` to this repo and `IMAGENET_ROOT` to the shared
  ImageNet ImageFolder; override either by env var.

## Synthetic stress suite (Tables 1, 3; Figs 1, 2)
- Run: `sbatch slurm/infodfa_multioutput_noise_sweep.sbatch` (128 cells = 4 regimes
  x 4 n_train {512,1024,2048,4096} x 4 label-noise {0,0.1,0.2,0.4} x 2 input-noise
  {0.05,0.15}; methods bp/dfa_random/fa_random/ndfa_random/ndfa_random_kronecker/
  drtp_random/vnc/nmnc; 5 seeds; feedback ranks {0,1,2,4,8}; hidden 256-128; 14 epochs;
  lr 0.08; natural-damping 0.3).
- Honest model selection: `analysis/reanalyze_synthetic_honest_selection.py` reports
  per-regime accuracy under test-selected vs fixed-full-rank vs leave-one-seed-out
  (LOSO) rank selection, plus seed-level paired tests. The three schemes agree to
  <0.1pp (full-rank feedback is essentially always best).
- Stat tests (Table 7): `analysis/compute_infodfa_statistical_tests.py`.

## Tuned-BP control (§5.4)
- Run: `sbatch slurm/infodfa_bp_tuning_synthetic.sbatch` (BP only, lr grid
  {0.02,0.04,0.08,0.16,0.32} at every cell).
- Aggregate: `analysis/aggregate_bp_tuning.py` (tuned BP per cell via LOSO over lr;
  seed-level paired test vs best conditioned).

## KFAC-DFA control (related work)
- Run: `sbatch slurm/infodfa_kfac_control_synthetic.sbatch` (adds
  `ndfa_random_kronecker_bp`: K-nDFA with a BP-error left factor instead of DFA-error).
- Aggregate: `analysis/aggregate_kfac_control.py` (K-nDFA minus KFAC-DFA(bp) = +0.01pp mean).

## Local-rule baselines (sign-symmetry, §6)
- Run: `sbatch slurm/infodfa_localrule_baselines_synthetic.sbatch` (adds `fa_sign`).

## Vision MLPs and ColoredMNIST (Tables 4, 12)
- Vision noisy-label sweep: `run_dfa_nmnc_comparison.py` (slurm `infodfa_nmnc_comparison.sbatch`).
- ColoredMNIST: `run_dfa_coloredmnist.py` (slurm `infodfa_coloredmnist.sbatch`).

## Reviewer controls (Table 10)
- Run: `sbatch slurm/infodfa_controls.sbatch` (BP/BP+L2/BP+label-smoothing/BP+early-stop/
  DFA/DFA+norm-match/nDFA/K-nDFA at one nuisance cell and one Fashion-MNIST cell, 5 seeds).

## Capable-model whitening vs norm-match (§5.4)
- Run: `sbatch slurm/infodfa_capable_normmatch.sbatch` (CIFAR-100 convnet, methods
  bp/local_loss/dfa_random/dfa_random_normmatch/ndfa_random/ndfa_random_kronecker;
  Adam; per-method lr {bp,dfa,normmatch 3e-4; ndfa 1e-4}; channels 64-128-256; 40 epochs;
  feedback-scale 0.3; 5 seeds x 5 feedback-seeds).
- Aggregate: `analysis/aggregate_capable_normmatch.py` (nDFA 23.6 vs DFA+norm-match 13.1).

## CIFAR-100 convnet (Table 8)
- Run: `sbatch slurm/infodfa_hard_cifar100_confirm.sbatch`.

## ImageNet-100 substitution-depth boundary (Table 6)
- Strong-form sweep: `sbatch slurm/infodfa_imagenet_strongform.sbatch` (13 configs:
  BP + {raw block-DFA, diagonal nDFA, full-covariance nDFA} x 4 depths
  {layer4, layer3+4, layer2+3+4, all}; pretrained ResNet-18; `--dfa-norm unit`
  (no BP-norm oracle); `--whiten-mode {diag,full}`; 90 epochs; lr 0.1; image 176).
- Deep-depth lr check for full-cov: `sbatch slurm/infodfa_imagenet_sf_lrcheck.sbatch`
  (full-cov at layer2+3+4 and all, lr {0.01,0.03}).
- Aggregate: `analysis/aggregate_imagenet_strongform.py`.

## Theory figure (Fig for §3.1)
- Self-contained (no external data): the `_theory_sim` / `make_theory_conditioning`
  routines in `drafts/Info-DFA/scripts/make_iclr_figures.py`. Panel C uses
  `task_in_high_var` to place the target in low- vs high-variance eigendirections.

## Regenerating all figures
```
INFODFA_RESULTS=<results dir> python drafts/Info-DFA/scripts/make_iclr_figures.py
```
`INFODFA_RESULTS` / `INFODFA_FIGURES` override the default paths; the script is
headless-safe (Agg backend).
